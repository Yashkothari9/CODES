the first half of the queue with the second half of the queue. Only a stack can be

used as an auxiliary space.

 IF YOU USE ANOTHER CONSTRUCT OTHER THAN MENTIONED YOU WILL GET A 0 FOR THE PROBLEM.

Output Format
